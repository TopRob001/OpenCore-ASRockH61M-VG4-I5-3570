# OpenCore EFI for ASRock H61M-VG4 + i5-3570 (Ivy Bridge)

This is an OpenCore EFI configuration configured for my PC:

## Hardware

* Motherboard: ASRock H61M-VG4
* CPU: Intel Core i5-3570 (Ivy Bridge)
* GPU: AMD Radeon RX 580

## Tested macOS Version

I successfully used this EFI to install and run macOS Mojave (10.14).

## Known Issues

### USB Ports

Currently, only two rear USB ports are working:

* Upper-left rear USB port
* Lower-right rear USB port

I was unable to properly map and configure the remaining USB ports.

## Notes

This EFI is provided as-is and was created specifically for my hardware configuration. Compatibility with other systems is not guaranteed.

Use at your own risk.
